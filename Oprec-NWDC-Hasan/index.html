Front-end Vs Back-end

Front-end

front-end adalah bagian tampilan website yang mana fokusnya adalah menampilkan isi dari website dan task-task yang diberikan oleh back end tampilan pada website juga harus mengutamakan aspek UI/UX. untuk menjadi seorang front end developer dibutuhkan skill-skill seperti HTML, css, JavaScript, RESTful Service, dan API

Back-end

Back-end adalah perintah-perintah yang dijalankan dibalik front end dan yang akan ditampilkan pada front end. back end ini juga bertanggung jawab atas server, database, dan manajemen konten yang ada pada suatu website. untuk menjadi seorang back end developer dibutuhkan skill bahasa pemrograman seperti JavaScript, PHP, Python, dan Ruby untuk bahasa yang dipakai di server
